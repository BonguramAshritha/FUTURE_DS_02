# FUTURE_DS_02

## Customer Retention & Churn Analysis

### Tools Used
- Power BI
- CSV Dataset

### Dashboard Includes
- Total Customers
- Churn Customers
- Contract vs Churn
- Payment Method vs Churn
- Churn Distribution
- Customer Tenure Analysis

### Key Insights
- Month-to-month contract customers have the highest churn.
- Long-term contract customers churn less.
- Most customers are retained.
- Customers with short tenure are more likely to churn.
